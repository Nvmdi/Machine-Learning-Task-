# Prediksi Konsumsi Bahan Bakar Kendaraan

## Deskripsi
Project ini bertujuan untuk memprediksi konsumsi bahan bakar kendaraan berdasarkan spesifikasi teknis menggunakan metode regresi.

## Dataset
Dataset berisi data kendaraan tahun 2015–2024 yang mencakup:
- Merek kendaraan
- Tahun model
- Kapasitas mesin
- Jumlah silinder
- Jenis bahan bakar
- Konsumsi bahan bakar

## Metode
Model yang digunakan:
- Linear Regression
- Random Forest
- Gradient Boosting

## Evaluasi Model
Model dievaluasi menggunakan:
- R2 Score
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)

## Tools
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Cara Menjalankan
1. Upload dataset
2. Jalankan notebook
3. Lihat hasil prediksi
